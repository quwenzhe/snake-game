# Snake Game

A classic Snake game built with TypeScript.

## How to Play

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the game:
   ```bash
   npm run dev
   ```

## Controls

- `w` - Move up
- `s` - Move down
- `a` - Move left
- `d` - Move right
- `r` - Restart game
- `q` - Quit game

## Game Rules

- Use the snake to eat the food (`*`)
- Each food eaten increases your score by 1
- Avoid hitting the walls or your own body
- Game ends when you collide with a wall or yourself
